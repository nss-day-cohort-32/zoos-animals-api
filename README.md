# Instructions

## Version
Make sure you have `dotnet 2.2.*`. Use `dotnet --version` in your console to check.

## API
1. Clone down this repo
1. Use the `animalsdb.sql` to create and seed your database.
1. Open the `AnimalsAPI.sln` in Visual Studio.
1. Build and run your application.

## Tests
1. Open the Test Explorer.
1. Click `Run`.
